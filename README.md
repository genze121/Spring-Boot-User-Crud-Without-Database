<h1> Spring-Boot-User-Crud-Without-Database </h1>

## Following features

> CRUD Operation
- C -> CREATE
- R -> READ
- U -> UPDATE
- D -> DELETE

## Technology stack
- Java 8+
- Spring Boot
- STS IDE
- Maven Build Tool
- Postman Client for Testing API
- Embedded Tomcat Server
